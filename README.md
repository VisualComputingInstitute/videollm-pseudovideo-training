# How Important are Videos for Training Video LLMs?

[[`Project Page`](https://visualcomputinginstitute.github.io/videollm-pseudovideo-training/)]

# Overview

This repository contains the code for our CVPR 2025 Workshop on Video LLMs submission, ["How Important are Videos for Training Video LLMs?](https://arxiv.org/pdf/2506.06928).
It provides a simple framework for constructing pseudovideo question-answer samples from datasets of annotated images.
It also provides a training pipeline based on the [LongVU work](https://github.com/Vision-CAIR/LongVU), with inference on the [TVBench dataset](https://github.com/daniel-cores/tvbench).
